Lista de comandos GIT

Comando Desripción
git init Inicializa la carpeta que será un repositorio
git status Muestra los cambios registrados
git log Despliega los commits realizados
git config --global user.name "Nombre" Guarda el nombre
git config --global user.email "Correo" Guarda el correo
git add -A Añade los cambios
git commit -m "Mensaje" Hace el commit
