Lista de comandos GIT

Comando Desripción
git init Inicializa la carpeta que será un repositorio
git status Muestra los cambios registrados
git log Permite visualizar los commit realizados
git log>archivo.txt Guarda los commits en un archivo de texto
git config --global user.name "Nombre" Guarda el nombre como variable global
git config --global user.email "Correo" Guarda el correo como variable global
git add -A Coloca los archios en el área de Stagging
git commit -m "Mensaje" Coloca los archivos en el área final del repositorio
git push origin master Coloca los archivos en el servidor
git checkout id_commit Se regresa al commit especificado
git checkout nombre_rama Se cambia hacia otra rama
git branch Imprime la rama de git sobre la que se está trabajando
git branch nombre_rama Crea una nueva rama
git push origin nombre_rama Sube los cambios de la otra rama al repositorio
git merge nombre_rama Se debe de ejecutar desde la rama master. Junta las dos ramas
